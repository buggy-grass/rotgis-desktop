RotGIS Desktop

RotGIS Desktop is a cross-platform desktop GIS application template built with Electron, TypeScript, React, Shadcn UI, Lucide React Icons, and Webpack. The application supports 2D maps using OpenLayers and 3D point cloud & mesh visualization using Potree.
